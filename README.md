# プロダクトゴール
### 自分が設定した目標が画面に大きく表示されて、達成すると次の目標が表示されるようにする

# スプリント３ですること
1.拡張機能を追加し、アプリについて確認する

2.修正点をissueに書き、新しいブランチで開発する。AIに修正させる。

3.Go liveで確認し、バグなどがないか確認する

4.変更をステージし、pull リクエストを作成する。

5.また新しいイシュー、ブランチを作成し、同じ工程を繰り返していく

# プロダクトバックログ

１.　目標設定機能を作る

２．達成率が見れる機能を作る

３.　目標を達成したら、達成した目標が消えるようにする機能を作る

## 〇デザイン・配置
色は明るい色味やオレンジを使う。

色で状態を表現
🟢 計測中
🔵 完了
🔴 一時停止

配置は真ん中に大きく目標を表示する感じで、達成率が分かるようにする機能は下の方に表示する。
